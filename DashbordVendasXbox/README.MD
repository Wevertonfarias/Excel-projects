# 📈 Métricas do Dashboard

O painel apresenta os seguintes indicadores:

### 💰 Receita Total
Valor total gerado pelas vendas de assinaturas.

### 👥 Total de Assinantes
Número total de clientes com assinatura ativa.

### 🧾 Ticket Médio
Valor médio gerado por venda.

### 📅 Receita Mensal
Gráfico mostrando o desempenho das vendas ao longo do ano.

### 📊 Receita por Tipo de Assinatura

Comparação entre planos:

- Mensal
- Trimestral
- Anual

### 🔄 Renovação Automática

Indicador de comportamento dos clientes:

- Assinantes com renovação automática ativa
- Assinantes com renovação desativada
