# 📊 Excel Projects

Repositório com projetos desenvolvidos em **Microsoft Excel** voltados para **análise de dados, dashboards e automação de relatórios**.

A ideia deste repositório é reunir diferentes projetos práticos utilizando Excel para estudo e portfólio, aplicando conceitos de:

- 📈 Visualização de dados
- 📊 Dashboards interativos
- 🧮 Fórmulas e funções avançadas
- 🔎 Análise exploratória
- 📑 Organização e tratamento de dados

Cada projeto possui sua própria pasta com:
- Arquivo Excel
- Imagens do projeto
- README explicando o funcionamento
